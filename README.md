# 🏆 Koora Gamers Hub

منصة اجتماعية خفيفة وعصرية تجمع بين مجتمع الجيمرز (Gamers) والتغطية الحية والشاملة لأحداث ومباريات **كأس العالم 2026**.

## 🚀 مميزات المنصة
* **الكل في مكان واحد:** دمج بين تفاعلات الجيمرز وأخبار المونديال.
* **تصميم عصري ونظيف:** واجهة بيضاء مريحة للعين، متجاوبة بالكامل مع الهواتف الذكية (Mobile-First).
* **توقيت حي ودقيق:** عداد تنازلي بالثانية لانطلاق البطولة ومواعيد المباريات.
* **مستودعات إعلانية منسقة:** مساحات مجهزة لدمج الإعلانات (Ad Slots) دون التأثير على تجربة المستخدم.

## 🛠️ التقنيات المستخدمة حالياً
* HTML5 / CSS3 (Custom Borders & Fluid Layouts)
* JavaScript (Live Countdown Timer)
